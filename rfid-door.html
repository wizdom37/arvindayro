Arduino Base RFID lock System

This project i develop is a keyless doorlock using RFID, arduino and vb.net.

unlike the my other door lock project this has a external system that i develop using vb.net

unlike my other RFID based door lock, this project has a external system using vb.net where you can register the user and

track who access the door every time.

i work for 4 months in this project before giving this to my client.

Characteristics of the system

To start the process of the system the administator will connect the arduino

to the software i develop and scan the ports and the baud rate of the arduino

Once it is connected we can now access the Arduino using the software i develop

In order for the user to access the administrator will register his or her data

and the RFID tag after the administrator register the user's data.

The user will only scan the RFID tag to the scanner and the system will scanner

scanner and the system will check if that data is registered or not

If it is registered the the green light will prompt and the buzzer will beep

the LCD text will say door open and the door will now unlock and it will be

lock after the short period of time or the user will lock it in the other side of

the door by pressing the push button

If it is not registered the red light will prompt and the buzzer will buzz

so loud and the LCD text will say door close and the door will remian

lock because the user is not yet registered or the user's data has been deleted

The administrator can track who access to the door in the history logs
