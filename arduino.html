Arduino Base RFID lock System

This project i develop is a keyless doorlock using RFID for my thesis.

The system can only be access by registering the RFID.

This is a fully stand alone arduino hardware system.

Only the staff of the office and the campus director of the campus can access the system.

i work for more than 1 year in this project before selling to my Research Instructor.

Characteristics of the system

To start the process of the system the admin of the system must have a unique

card and it will serve as the master key. To registrar the RFID the admin of the

system must tap the master key to the RFID scanner and the LCD text will be program

made Add/Remove a tag. Once the user tap the RFID tag will be added to the database

LCD text will be Tag added, however if the user taps the already registered tag

during the program mode the Arduino program will delate the tag and the LCD text

will be tag removes. To start the process of unlocking the door the user will

tap the RFID card to the scanner, and it will send a signal to the Arduino to

determine if it required , and if it is registered the LCD test will be access

granted the green LED will lights up the Buzzer will beep and the servo motor

will rotate, when the user is already inside the office there are two buttons

near at the door and the red button serve as unlocking the door and the blue

button is unlock the door however if the user will tap an unregistered the same

process will be occurred
