Respository System

This is latest project for my client using vb.net for their thesis.

The system stored their research of the students and teachers as a PDF file, and provided interface for each user of the system especially for the administrator of the system.

research results would be generated by the system in PDF form and able to print or download with an interface for other department to be able to login and view.

This system has a admin side and a user side.

Other students and teachers had the ability to view their data and able to add notes.

This system also have a History logs for all the user able to identify who log in for this date and time for security purposes.

I worked on this system for a 2 months before giving this to my client.

Technologies:
  • - Visual Studio 2012
  • - Adobe Acrobat 8 Professional
  • - AxAcroPDF - PDF generator
  • - Microsoft Access
